Design System

A small design system for a portfolio website. Built utilizing modified Bootstrap components.

This design system was created for HCI472 at DePaul University.

It was designed with Adobe XD, and built utilizing the Bootstrap grid and customized Bootstrap components. When I envisioned this design system, I wanted to create a simple layout for the kind of portfolio typical of an HCI or UX student. That is, a layout that allows for maximum flexibility, as UX case studies often vary in content and form.

I chose to stick with a simple color palette in order to highlight the varied visuals inherent on a portfolio site.

I chose not to base this portfolio on my personal brand, which I defined at the beginning of the quarter as: playful, forward-thinking, creative, and strategic-- a mix of futurism and nostalgia. It's not that I don't think those adjectives apply to myself as a writer, designer, and researcher in some sense. It's mostly that as I took this class, I realized that my design sensibility is always shifting and always changing (because I still [always] have much to learn), so I used this as an experiment to do something outside of my normal in-your-face kindergarten-colored aesthetic.

For this project, I truly wanted to focus on the underlying structure of the system, so I did not want myself or the user to become distracted by a potentially overwhelming aesthetic.

The result is a simple system that can be used in many ways, that allows visuals to shine all on their own.
